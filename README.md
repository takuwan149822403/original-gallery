# オリジナルギャラリー 画面設計書

## 1. 文書情報

- 文書名: オリジナルギャラリー 画面設計書
- 対象システム: オリジナルギャラリー
- 作成目的: Flask + SQLAlchemy + SQLite を用いた画像管理Webアプリの画面仕様整理
- 対象バージョン: Version 1

---

## 2. システム概要

本システムは、画像・タイトル・説明文を登録し、一覧表示および詳細表示、削除を行う Web アプリケーションである。  
画像本体は DB に保存し、Flask が画像レスポンスを返却する。

---

## 3. 画面一覧

| 画面ID | 画面名 | テンプレート | URL | 概要 |
|---|---|---|---|---|
| SCR-001 | 一覧画面 | `main_page.html` | `/` | 登録済み画像の一覧表示、登録画面遷移、全件削除 |
| SCR-002 | 登録画面 | `register.html` | `/pictures/new` | 画像・タイトル・説明文の新規登録 |
| SCR-003 | 詳細画面 | `picture_detail.html` | `/pictures/<picture_id>` | 指定画像の詳細表示、1件削除 |
| SCR-004 | 4xxエラー画面 | `4xx.html` | 各種4xx発生時 | 入力不備・URL不正などのエラー表示 |
| SCR-005 | 5xxエラー画面 | `5xx.html` | 各種5xx発生時 | サーバ内部エラー表示 |

---

## 4. 画面遷移

```text
一覧画面 (/)
 ├─ [画像を登録] → 登録画面 (/pictures/new)
 ├─ [画像カードクリック] → 詳細画面 (/pictures/<picture_id>)
 └─ [全件削除] → 一覧画面へ再表示

登録画面 (/pictures/new)
 ├─ [登録成功] → 詳細画面 (/pictures/<picture_id>)
 └─ [一覧に戻る] → 一覧画面 (/)

詳細画面 (/pictures/<picture_id>)
 ├─ [この画像を削除] → 一覧画面 (/)
 └─ [一覧に戻る] → 一覧画面 (/)

4xx/5xx エラー画面
 └─ [一覧に戻る] → 一覧画面 (/)

```

## 5. 画面詳細

## 5.1 共通レイアウト
* 画面全体は .page クラスで中央寄せ表示する
* 最大幅は 960px
* 背景色は淡いグレー系 (#f5f7fb)
* 本文文字色は濃いグレー (#1f2937)
* タイトル、カード、フォーム、エラーページで共通デザインを使用する

## 5.2 共通パーツ
| パーツ名    | CSSクラス              | 用途              |
| ------- | ------------------- | --------------- |
| ページ全体   | `.page`             | コンテンツの横幅制御と余白設定 |
| ヘッダ     | `.page-header`      | 各画面タイトル領域       |
| 画面タイトル  | `.page-title`       | 画面名表示           |
| 補足文     | `.page-subtitle`    | 画面説明表示          |
| 操作ボタン群  | `.actions`          | ボタン・リンクの横並び表示   |
| 主ボタン    | `.button`           | 主操作             |
| 副ボタン    | `.button-secondary` | 戻る等の補助操作        |
| 危険操作ボタン | `.button-danger`    | 削除操作            |
| カード     | `.card`             | 白背景コンテンツ領域      |
| 空状態表示   | `.empty-state`      | データ0件時の表示       |

## 5.3 レスポンシブ対応
* 画面幅 640px 以下では余白と文字サイズを縮小する
* 一覧カードは grid で自動折返し表示する

## 5.4 main_page.html
画面構成
| 項番 | 項目名      | 種別      | 必須 | 表示/入力内容           | 備考                     |
| -- | -------- | ------- | -- | ----------------- | ---------------------- |
| 1  | 画面タイトル   | 表示      | -  | オリジナルギャラリー        | `.page-title`          |
| 2  | 補足文      | 表示      | -  | 登録した画像を一覧で確認できます。 | `.page-subtitle`       |
| 3  | 画像を登録    | ボタン/リンク | -  | 登録画面へ遷移           | `/pictures/new`        |
| 4  | 全件削除     | ボタン     | -  | 全画像削除             | POST送信                 |
| 5  | 画像一覧     | 一覧表示    | -  | 画像カードを繰返し表示       | `.gallery-list`        |
| 6  | サムネイル    | 画像      | -  | DB保存画像を表示         | `/pictures/<id>/image` |
| 7  | タイトル     | 表示/リンク  | -  | 画像タイトル            | 詳細画面へ遷移                |
| 8  | 登録日時     | 表示      | -  | `created_at`      | 補足情報                   |
| 9  | 空状態メッセージ | 表示      | -  | まだ画像が登録されていません。   | データ0件時のみ               |

## 5.5 register.html
画面構成
| 項番 | 項目名    | 種別      | 必須 | 入力/表示内容            | 備考                 |
| -- | ------ | ------- | -- | ------------------ | ------------------ |
| 1  | 画面タイトル | 表示      | -  | 画像登録               | `.page-title`      |
| 2  | 補足文    | 表示      | -  | 画像、タイトル、説明文を登録します。 | `.page-subtitle`   |
| 3  | 画像     | ファイル入力  | ○  | 画像ファイル             | `accept="image/*"` |
| 4  | タイトル   | テキスト入力  | ○  | 100文字以内            | `maxlength="100"`  |
| 5  | 説明文    | テキストエリア | -  | 2000文字以内           | `maxlength="2000"` |
| 6  | 登録     | ボタン     | -  | 登録実行               | POST送信             |
| 7  | 一覧に戻る  | ボタン/リンク | -  | 一覧画面に戻る            | `/`                |

## 5.6 picture_detail.html
画面構成
| 項番 | 項目名     | 種別      | 必須 | 表示内容              | 備考                     |
| -- | ------- | ------- | -- | ----------------- | ---------------------- |
| 1  | 画面タイトル  | 表示      | -  | 画像詳細              | `.page-title`          |
| 2  | 補足文     | 表示      | -  | 登録済み画像の内容を確認できます。 | `.page-subtitle`       |
| 3  | 画像      | 表示      | -  | DB保存画像            | `/pictures/<id>/image` |
| 4  | タイトル    | 表示      | -  | 画像タイトル            | `.detail-heading`      |
| 5  | 登録日時    | 表示      | -  | `created_at`      | `.detail-meta`         |
| 6  | 説明文     | 表示      | -  | `description`     | 未設定時は既定文表示             |
| 7  | この画像を削除 | ボタン     | -  | 1件削除              | POST送信                 |
| 8  | 一覧に戻る   | ボタン/リンク | -  | 一覧画面へ遷移           | `/`                    |
